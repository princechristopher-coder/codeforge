Introduction

HTML (HyperText Markup Language) defines the structure of web pages using elements like headings, paragraphs, and links.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
